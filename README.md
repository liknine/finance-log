# Finance Log

Локальный React/Vite проект для FINANCE LOG.

## Запуск

```bash
npm install
npm run dev
```

После запуска открой ссылку из терминала, обычно `http://localhost:5173`.

## Структура

```txt
src/
  App.tsx
  styles.css
  data/mockData.ts
  utils/finance.ts
  components/Layout.tsx
  components/UI.tsx
  pages/Dashboard.tsx
  pages/Shipments.tsx
  pages/NewShipment.tsx
  pages/Templates.tsx
  pages/Tools.tsx
  pages/Settings.tsx
```
